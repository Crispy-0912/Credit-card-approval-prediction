# Credit Card Approval Prediction

A machine learning web application that predicts whether a credit card application will be approved.

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Scikit-learn
- Random Forest

## Features

- User-friendly web interface
- Predicts approval based on user details
- Machine learning model using Random Forest

## Run the Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Train the model:

```bash
python model.py
```

Run the application:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000/
```
